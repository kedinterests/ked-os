# KED-OS

An operating system for KED Interests — web development and digital solutions for the oil & gas energy sector.

## What This Is

KED-OS is a structured environment for managing projects, teams, standards, and processes for web development work serving the oil & gas energy sector with deep expertise in mineral rights, drilling operations, and industry-specific solutions.

## Core Idea

- **CLAUDE.md:** Project rules, team structure, model routing, skill organization
- **memory/:** Persistent context across sessions (people, company, glossary, radar)
- **skills/:** Specialized workflows (code verification, security audit, industry news prep, etc.)
- **core/:** Technical standards and patterns (Astro, Discourse, Vibe, the KED stack)
- **internal/:** Project folders, templates, and ongoing work

## Key Principles

- **No WordPress.** Ever. KED uses Astro, Discourse, hand-coded Vibe sites.
- **Modern tech stack.** JavaScript/TypeScript, Node.js 18+, performance-first.
- **Specialized for energy sector.** Deep domain knowledge, not generic web dev.
- **Fast shipping.** Pragmatism over perfection; ship over optimize.
